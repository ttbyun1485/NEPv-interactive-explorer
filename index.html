THE CONCEPT

Understanding the NEPv Setup

In a standard linear eigenvalue problem, $Av = \lambda v$, the matrix $A$ is fixed. An eigenvector $v$ is a "special" vector that, when transformed by $A$, only changes in scale but not direction.

In a Nonlinear Eigenvalue Problem with eigenvector dependency (NEPv), the matrix $A(v)$ is no longer static but changes based on the vector $v$ itself. Nevertheless, we are still hunting for those unique vectors $v$ that remain perfectly aligned with their output after the transformation.

01
The Dynamic Matrix $A(v)$

For the example below, we define the matrix $A$ as follows:

$$ A(v) = \begin{pmatrix} x^2 & 1 \\ 1 & y^2 \end{pmatrix} $$
